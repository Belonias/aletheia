A Proposed Solution: Aletheia

Leverage our natural method for fake detection: reputation

Consider for a moment what normal humans (not nerds) do when considering what is real.

  • Say I were to, right now, make an outrageous claim.
  • A claim that Ben here, in his off hours, runs a pseudo-scientific website selling magic stones for people's pets.
  • It's true! He claims that amethyst cures Cancer and that quartz fights tooth decay.
    • How do you know what's real?
    • Show of hands: who would use AI for this problem?
    • Friends, family, WHOIS, network trickery, etc.

How it Works

Now with diagrams!

So that's the problem, let's talk about a solution.

[ Signing ]


Here's what a file typically looks like


Aletheia pulls the body of the file out and generates a signature of it with your private key


Tack the signature onto the header,
along with where to find the public key


The resulting file is a little bigger,
but now contains the origin

[ Verification ]


Extract the key URL and signature from the header


Fetch the key from that URL.


Use the key to verify the signature against the file body
